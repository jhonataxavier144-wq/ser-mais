# SER+ Plataforma de Cursos

Plataforma de cursos gratuita focada em desenvolvimento pessoal e profissional.

## Tecnologias
- HTML/CSS/JavaScript puro
- Firebase Authentication (login Google + e-mail)
- Firebase Firestore (banco de dados)
- Vercel (hospedagem)

## Estrutura
```
ser-mais/
├── index.html          → Redireciona para login ou dashboard
├── login.html          → Tela de login/cadastro
├── dashboard.html      → Área do aluno
├── vercel.json         → Configuração da Vercel
└── js/
    └── firebase-config.js
```

## Deploy
1. Conecte este repositório na Vercel
2. Clique em Deploy
3. Acesse o link gerado pela Vercel
